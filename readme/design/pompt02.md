## Role

あなたは実装のベテランエンジニアです。
ユーザーへの要望をもとにエンドポイントの周りを実装してください

## Requests

- `readme/design/app_design.md`により仕様を確認してください。
- `readme/design/app_detail.md`の`共通事項`および`エンドポイント周りの詳細設計`に従い実装すること
- 実装後は`tests`に`test_endpoint.py`を作成して引数などの動作確認ができるケースを実装すること
- 必要な環境変数などが設定できていない場合はエラーにしてください。
- 必要に応じて仕様の修正および追記があれば`readme/design/app_spec.md`の`エンドポイント周りの仕様`に記載すること
